# Amarelo Sincronizador de Legendas

Um programa moderno para sincronização automática de legendas com vídeos.

## Características

- 🔄 Sincronização automática de legendas
- 🎨 Interface moderna e intuitiva
- 🌍 Suporte a múltiplos idiomas
- 📝 Formatação inteligente de legendas
- 🔒 Proteção contra modificações durante a análise

## Requisitos

- Windows 10 ou superior
- Python 3.8+ (para desenvolvimento)

## Instalação

1. Baixe o arquivo executável
2. Execute o instalador
3. Siga as instruções na tela

## Uso

1. Selecione a pasta com vídeos e legendas
2. Configure as preferências de legenda
3. Clique em "Iniciar Sincronização"
4. Aguarde a conclusão da análise

## Idiomas Suportados

- Português (Brasil)
- Português (Portugal)
- English (US)
- English (UK)
- Français
- Deutsch
- Español
- 日本語
